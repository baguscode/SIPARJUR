Tentang Aplikasi

SIPARJUR dirancang untuk membantu calon mahasiswa menentukan pilihan program studi kuliah terbaik berdasarkan indikator minat dan kecenderungan kompetensi akademik mereka.

Metode Inferensi

Sistem ini menerapkan penalaran Forward Chaining, di mana fakta jawaban pengguna atas indikator minat dicocokkan ke dalam basis aturan untuk melahirkan kesimpulan prodi.

Keunggulan Cloud

Mengadopsi teknologi Serverless Dynamic Database terintegrasi dengan Google Sheets API, mempermudah pakar mengubah parameter bobot nilai kapan saja.
